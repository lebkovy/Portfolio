Mobile Apps

My mobile applications are developed with Google's framework called Flutter which is a cross-platform framework that allows me develop apps both on Android and iOS systems. It uses Dart language that is very fast and client-optimized for mobile apps.

My apps often based on functions provided by Firebase which includes several very useful modules for developing apps.

Latest project

iValut is an application that allows you to calculate your gains or losses on currency operations. An algorithm compares the current exchange rate of a given currency and calculates the difference of your balance. If the balance is positive, you have gained the calculated amount - if not, you have lost the indicated amount.
